# Showey App <a name="about-project"></a>

**Showey App** Showey is a web app that is built with Vite + React. It fetches movies and TV showes from [OMDb API](https://www.omdbapi.com/). It allows users to search for thier favourite shows. Users can add showes to watched list and rate them.
